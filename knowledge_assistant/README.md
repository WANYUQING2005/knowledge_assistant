# ⚙️ 主配置模块 (knowledge_assistant)

  <p><strong>知识助手项目的核心配置与系统管理中枢</strong></p>

## 📋 模块简介

主配置模块是整个Django项目的核心控制中心，包含项目全局设置、URL路由配置、部署入口以及系统级工具。它协调其他所有模块的工作，提供统一的配置管理和系统服务。

## 🔑 核心功能

- **⚙️ 项目配置**
  - 数据库连接设置
  - 中间件与应用注册
  - 静态文件与媒体文件配置
  - 安全与认证设置

- **🔀 全局路由**
  - API端点注册与分发
  - 权限与认证中间件
  - 请求处理与响应格式化
  - 静态资源路由

- **🗃️ 向量存储**
  - FAISS向量数据库配置
  - 向量检索与相似度计算
  - 嵌入模型集成
  - 缓存与性能优化

- **🚀 部署支持**
  - WSGI/ASGI服务配置
  - 静态文件收集与处理
  - 环境变量管理
  - 日志与监控设置

## 📁 目录结构

```
knowledge_assistant/
│
├── __init__.py          # 包初始化
├── settings.py          # Django项目全局设置
├── urls.py              # 全局URL路由配置
├── asgi.py              # ASGI部署入口
├── wsgi.py              # WSGI部署入口
├── vector_store.py      # 向量存储实现
├── sql/                 # SQL脚本和数据库相关
└── __pycache__/         # Python缓存文件
```

## 🛠️ 主要配置文件

### settings.py
此文件定义了Django项目的所有关键设置，包括：
- 已安装应用列表
- 中间件配置
- 数据库连接参数
- 认证后端设置
- 静态文件与媒体文件路径
- 国际化与本地化设置
- 安全设置与CORS配置

### urls.py
此文件负责所有URL路由的注册与映射：
- API路由注册
- 静态资源路由配置
- 管理后台路由
- 认证路由

### vector_store.py
此文件实现向量存储与检索功能：
- FAISS向量数据库初始化
- 文本嵌入与向量化逻辑
- 相似度搜索算法
- 结果排序与过滤

## 🔗 与其他模块的关联

- 为**所有模块**提供全局配置和服务
- 注册并连接所有模块的URL路由
- 提供向量存储服务给**知识库**和**RAG模块**
- 管理整个系统的认证、权限和安全策略
