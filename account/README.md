## 📋 模块简介

账户管理模块负责处理用户的注册、登录、身份验证以及个人资料管理等功能。它是知识助手项目的基础模块，为整个系统提供用户身份识别和权限管理支持。

## 🔑 核心功能

- **👥 用户管理**
  - 用户注册与账户创建
  - 用户登录与身份验证
  - 基于令牌的认证系统
  - 密码重置与账户恢复

- **📝 个人资料**
  - 个人基本信息管理
  - 头像和个性化设置
  - 用户偏好存储
  - 活动历史记录

- **🔒 权限控制**
  - 基于角色的访问控制
  - 用户组与权限管理
  - 安全策略实施

## 🔌 API 接口

| 接口路径 | 方法 | 描述 | 权限 |
|---------|------|------|------|
| `/api/account/register/` | POST | 新用户注册 | 公开 |
| `/api/account/login/` | POST | 用户登录 | 公开 |
| `/api/account/profile/` | GET | 获取个人资料 | 需认证 |
| `/api/account/profile/` | PUT | 更新个人资料 | 需认证 |
| `/api/account/password/` | PUT | 修改密码 | 需认证 |

## 📊 数据模型

### 用户模型 (User)
- `username`: 用户名
- `email`: 电子邮箱
- `password`: 密码（加密存储）
- `is_active`: 账户状态
- `date_joined`: 注册日期

### 个人资料模型 (Profile)
- `user`: 关联的用户（一对一）
- `bio`: 个人简介
- `avatar`: 头像图片
- `phone`: 电话号码
- `website`: 个人网站
